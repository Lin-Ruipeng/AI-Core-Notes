### 为什么这份 Prompt 对你至关重要？

1. **暗中补齐“线性代数”**：端侧 AI 部署（无论是神经网络的前向传播，还是传统的卡尔曼滤波）本质上全是矩阵运算。我在 Prompt 的大纲要求里帮你加上了“线性代数”，这是微积分和概率统计之间必不可少的桥梁。
2. **引入 HPC 核心思维（数值稳定性）**：在纸上算方差 $D(X) = E(X^2) - [E(X)]^2$ 是绝对正确的，但在 C++ 里如果直接这么写，当数值非常相近时，会因为浮点数相减导致严重的精度丢失（灾难性取消）。LLM 看到这份 Prompt 后，在教你统计学时，就会强制让你用 **Welford 算法** 这种工业级的数值稳定算法来写 C++ 代码。
3. **技术栈的闭环**：Prompt 中特别提醒了 LLM 你已经具备“智能指针、移动语义、多线程”的能力。在后续的作业中，LLM 大概率会让你写出诸如：“请用多线程并行的蒙特卡洛方法（Monte Carlo）求解某个复杂分布的积分，并用智能指针管理内存”。这样一来，你之前学过的所有 C++ 知识都将在数学算法的实战中被彻底激活。
