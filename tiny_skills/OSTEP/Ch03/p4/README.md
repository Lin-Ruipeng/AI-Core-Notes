# `p4` 程序说明

## 快速开始

一键运行

```bash
# 默认不保存输出文件
bash ./p4.sh
# 保存输出文件
bash ./p4.sh save
```

## 结果说明

程序会创建文本文件 `p4.output` 并把子程序的输出结果写入。

本程序主要是展示了采用 `fork()` 和 `execvp()` 接口如何实现输出的重定向继承。

原理解释：首先是 `fork()` 会复制自己一份, 然后在子进程中, 使用 `close` 和 `open` 之后可以把子程序自身的输出重定向到文件。
接着使用 `exec()` 就会把指定的子程序代码覆盖掉自身, 而由于前面进行了重定向操作, 所以这种重定向会被集成到子程序中, 所以子程序实现了输出重定向。
这里最关键的点在于 `exec()` 的逻辑是覆盖指定程序代码到自身上!
